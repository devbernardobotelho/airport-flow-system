# Airport Flow System

Sistema de gerenciamento de operações aeroportuárias focado em controle de pousos, alocação de slots de pista e gerenciamento de stands (portões e posições remotas).

## Objetivo

Este projeto simula um ambiente operacional de aeroporto, permitindo que companhias aéreas e operadores gerenciem:

- Voos (Flights)
- Slots de pista (Runway Slots)
- Stands (Gate / Remote)
- Companhias aéreas (Airlines)

O foco é garantir organização, evitar conflitos e priorizar voos críticos (ex: emergências).
